Flake8 Plone API plugin
=======================
Checks for code usages that can be replaced with Plone API method calls.

Install
-------
Install with pip::

    $ pip install flake8-plone-api

Requirements
------------
- Python 2.6, 2.7, 3.3, 3.4
- flake8
